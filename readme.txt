* Under BerlinClock.cs in the main method , 
Different type of inputs are given to the formatToBerlinClock method and the output in shown in console. 

* If you run from the main method you can see the output for different inputs.

* You can pass your custom input by, 
calling the formatToBerlinClock method via berlinClockfinder object and pass the input as a parameter.

* The time input should be a valid standard time format or else you'll receive error message as "wrong input format"

* You can see the output in terminal by CTRL+F5.
